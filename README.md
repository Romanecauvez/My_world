# MY WORLD

Isometric 3D map modeling with CSFML

Build your dream world !

## Usage
```C
./my_world
```
## Tools
```C
- Add
- Dig
- Flat
```

## Key binding
```C
- ZQSD : Moove map
- ADD & SUBTRACT : Zoom and dezoom map
- A & E : Increase and decrease brush size
- Arrows : Rotate map
```
